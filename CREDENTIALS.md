# FreelanceKG - Учётные данные

## Пользователи

| Email | Пароль | Роль |
|-------|--------|------|
| admin@freelance.kg | admin123 | ADMIN |

## Требования к регистрации

- **Email**: валидный формат (example@domain.com)
- **Пароль**: от 6 до 100 символов
- **Полное имя**: обязательное поле
- **Телефон**: уникальный номер (формат: +996XXXXXXXXX)

## Доступ к сервисам

### Приложение
- **Frontend**: http://localhost:3000
- **Backend API**: http://localhost:8080
- **Swagger UI**: http://localhost:8080/swagger-ui.html

### База данных
- **Host**: localhost
- **Port**: 5433
- **Database**: freelance_kg
- **User**: postgres
- **Password**: postgres

## Роли в системе

| Роль | Описание |
|------|----------|
| USER | Обычный пользователь (заказчик и/или исполнитель) |
| ADMIN | Администратор (модерация, админ-панель) |

## Примечания

- Исполнитель = USER с профилем исполнителя (не отдельная роль)
- Профиль исполнителя создаётся автоматически при регистрации
- Настройка видимости в списке исполнителей: `/profile`
- Верификация исполнителя: `/admin/verifications`
- S3 не настроен - загрузка файлов использует mock URLs
